# 🤖 Interview AI

An AI-powered interview preparation platform built with React, Node.js, and Google Gemini API.

## ✨ Features
- 🔐 JWT Authentication with Token Blacklisting
- 📄 Resume Parsing & Skill Extraction
- 🤖 AI-Generated Technical & Behavioral Questions
- 📊 Match Score & Skill Gap Analysis
- 🗺️ 7-Day Personalized Preparation Roadmap
- 📥 ATS-Optimized Resume PDF Download

## ⚙️ Tech Stack
- **Frontend:** React.js, SCSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **AI:** Google Gemini API
- **PDF:** Puppeteer
